Page list:

  • /config: change configuration (SSID, password)
  • /restart: reboots the device
  • /status: show current status
  • /ls: see what's on the SPIFFS
  • /download?file=... download a given file from SPIFFS
  • /upload?file=... upload a given file to SPIFFS (`curl -F "@$PWD/file.html" @IP@/upload`)
